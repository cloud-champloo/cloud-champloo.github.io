コミュニティ立ち上げの背景

皆様はじめまして、JAWS-UG沖縄の副代表をしております西島と申します。

本年2月22日に開催いたしました初回会合にご参加いただいた皆様、ありがとうございました。

同会合におきまして、当コミュニティの必要性が認められたと感じましたので、早速コミュニティを立ち上げ、第1回の対外的イベントを開催したいと考えております。

趣旨としては変わらず、沖縄県内のデータセンター運営者/クラウド事業者/関連ユーザーグループを対象に、相互の人材交流や技術動向/情報交換、マーケット動向を相互交換し、沖縄のクラウド事業を盛り上げることを目的に、事業者間のコミュニティを作成することと考えております。

各事業者の皆様のご参加を心よりお待ちしております。

イベントの募集ページをこちらに作成いたしました。是非ご応募下さい。

コミュニティの構成

以下の様な構成要素で当コミュニティを構成したいと考えております。

  • 実行委員長：当面小生が担当させていただきます
  • 副実行委員長：将来いづれかのタイミングで、参加者の中より選出させていただければと思います
  • 協賛企業：法人単位で企画に賛同して下さる企業様を、協賛企業として登録させていただきます
  • コミュニティ参加者：個人単位で企画に賛同し、コミュニティ主催のイベントに参加して下さる方
  • 会計担当：当方が実行委員会を努めております ハッカーズチャンプルー実行委員会 で担当いたします

いづれの構成要素に尽きましても、登録費などは無料です。

また、各単発イベント/勉強会の方向性として、以下の2つの部会が内部にあることを想定しています。ただし現状では、誰がどの部会に所属する、などの細かい区分けはしません。

  • マーケティング部会：各企業の社内向け教育、および社外向けマーケティング/啓蒙活動
  • 技術部会：各企業の技術者間の情報交換/ハンズオンなどの開催

第1回クラウドチャンプルー　イベント企画概要

  • イベントの募集ページ
  • 参加費：無料
  • 目標集客人数：80-100人 （内訳:各企業様 4-5人×10 / エンドユーザ様 2人×10 / 行政関係者 10人 / その他県外からの聴講者など+α）

集客のターゲット

  • 「クラウドなんてよく分からん」という各協賛企業様社内の営業、もしくは技術者、そして意思決定権を持つ方々
  • 同じく「クラウドの意味が分からん」というプリセールス中などのエンドユーザ様
  • 県の経済動向に大きな影響力を持つ県職員の方々

イベントフォーマット：45分登壇者よりプレゼン＋15分ask the speakerの時間を取る

JAWS-UG沖縄のご紹介

http://jaws-ug.jp/bc/okinawa/

AWS User Group - Japan（通称JAWS-UG）は、Amazon Web Services（AWS）の利用促進や情報交換のためのユーザーグループです。

完全にコミュニティベースで運営されており、JAWS-UG沖縄はもちろん日本最南端のユーザーグループとして活動しています。

現在は年に2-3回の大きめのイベント/勉強会と、2014年は毎月無料でAWSを実際に試すことが出来る技術者向けのハンズオンを開催しております。JAWS-UG沖縄主催/関連のイベントは上記URLに紹介されておりますので、是非ご参加下さい。
